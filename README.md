# Framework

A PHP framework based on ZAPHYR repositories.

## System Requirements

You need `PHP => 8.1` but the latest stable version of PHP is recommended.

The framework is continuously tested on the following PHP versions:

- PHP 8.1
- PHP 8.2
- PHP 8.3
- PHP 8.4
- PHP 8.5

## Installation

It's recommended that you use [Composer](https://getcomposer.org/) to install the framework repository.

```console
composer require zaphyr-org/framework
```
